# Pink English Academy

Возможности:

-Добавление уроков (название + описание) 

-Просмотр списка всех уроков

-Добавление уроков в избранное

-Удаление уроков

-Сохранение данных в LocalStorage

-Имитация обновления списка уроков

-Форма обратной связи с отправкой данных

-Всплывающие уведомления об успехе и ошибках

#

Технологии: HTML5, CSS3, JavaScript, Fetch API, LocalStorage
